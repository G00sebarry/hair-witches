# 🧙‍♀️ Hair Witches: Color Rush

Промо-игра для салона красоты **Hair Witches** — 2D браузерный раннер в стиле кибернеон.

## Структура проекта

```
hair-witches/
├── index.html              # Точка входа
├── README.md               # Документация
├── css/
│   └── style.css           # Стили
├── js/
│   ├── core/
│   │   ├── audio.js        # 8-bit музыка и звуковые эффекты (Web Audio API)
│   │   ├── game.js         # Главный игровой цикл и управление состояниями
│   │   ├── input.js        # Обработка touch/mouse ввода
│   │   └── sprites.js      # Генерация SVG-спрайтов
│   ├── entities/
│   │   ├── player.js       # Игрок (ведьма) — жизни, щит, неуязвимость
│   │   └── objects.js      # Бонусы и враги — спавн, движение, коллизии
│   ├── screens/
│   │   ├── start.js        # Стартовый экран
│   │   ├── select.js       # Выбор ведьмы (3 темы)
│   │   ├── hud.js          # Очки, жизни, уровень
│   │   ├── gameover.js     # Экран Game Over с промокодами
│   │   └── levelup.js      # Переход между уровнями
│   ├── systems/
│   │   ├── particles.js    # Система частиц
│   │   ├── background.js   # Параллакс фон города
│   │   └── camera.js       # Тряска экрана, вспышки
│   └── utils/
│       ├── constants.js    # Все конфигурации игры
│       └── helpers.js      # Утилиты (canvas, коллизии, рендер)
└── assets/                 # Папка для будущих ресурсов
    ├── sprites/            # PNG спрайты (когда заменим SVG)
    ├── audio/              # Звуковые файлы
    └── fonts/              # Кастомные шрифты
```

## Как запустить

### Вариант 1: Просто открыть (НЕ работает из-за модулей)
Файлы загружаются через `<script>` теги, поэтому нужен локальный сервер.

### Вариант 2: VS Code + Live Server (рекомендуется)
1. Установи расширение **Live Server** в VS Code
2. Открой папку `hair-witches` в VS Code
3. Правый клик на `index.html` → **Open with Live Server**
4. Игра откроется в браузере

### Вариант 3: Через терминал
```bash
cd hair-witches
# Python
python3 -m http.server 8080
# или Node.js
npx serve .
```
Затем открой http://localhost:8080

## Геймплей

- **Управление:** Зажми экран/клик — ведьма летит вверх. Отпусти — падает.
- **Цель:** Собирай бонусы, избегай врагов, проходи уровни.

### Объекты
| Спрайт | Тип | Эффект |
|--------|-----|--------|
| 🧪 Зелье | Бонус | +10 очков |
| 💎 Кристалл | Супер-бонус | +50 очков |
| 🛡 Щит | Защита | Поглощает 1 удар (4 сек) |
| ❤ Сердце | Жизнь | +1 жизнь (макс 3) |
| ☠ Коробка краски | Враг | Летит прямо |
| ✂ Ножницы | Враг | Движется по синусоиде |
| 💣 Бомба | Враг (ур.2+) | Следит за игроком по Y |

### Уровни
1. **Улицы Неона** — целевой счёт 300, средняя сложность
2. **Токсичные Крыши** — целевой счёт 500, повышенная сложность, бомбы
3. **Огненный Финал** — бесконечный, максимальная сложность

### Промокоды (Game Over)
| Очки | Титул | Код | Скидка |
|------|-------|-----|--------|
| 0-499 | Цвет смылся! | START5 | 5% |
| 500-1499 | Продвинутая ведьма! | COLOR10 | 10% |
| 1500+ | Верховная ведьма! | SUPERWITCH | 20% |

## Выбор ведьмы

- 🔥 **ОГОНЬ** — пламенный стиль

## Настройка

Все параметры игры в файле `js/utils/constants.js`:
- Скорость, сложность, интервалы спавна
- Цвета и темы ведьм
- Промокоды и пороги очков
- Физика игрока

## Замена спрайтов на PNG

Когда будут готовы PNG-картинки, положи их в `assets/sprites/` и обнови `sprites.js`:
```js
// Вместо SVG-генерации:
function createPotion() {
  const img = new Image();
  img.src = 'assets/sprites/potion.png';
  return img;
}
```

## Интеграция на сайт

Игру можно встроить через iframe:
```html
<iframe src="game/index.html" width="100%" height="600" frameborder="0"></iframe>
```

## Технологии
- Vanilla JavaScript (ES6+)
- Canvas 2D API
- Web Audio API (8-bit музыка)
- SVG-спрайты (inline)
- Без зависимостей и фреймворков
