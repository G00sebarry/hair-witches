> Оставь немного пустого пространства слева (там будет хвост частиц от кисти).

## 🏙 Размеры фонов

Фоны — это **горизонтально тайлящиеся** полоски. Они скроллятся справа налево с разной скоростью (параллакс).

| Файл | Размер (px) | Скорость | Описание |
|------|-------------|----------|----------|
| `level1-far.png` | **1920×1080** | Медленная (×0.3) | Дальние здания, силуэты. Фиолетовое небо. |
| `level1-near.png` | **1920×1080** | Быстрая (×0.6) | Ближние здания с деталями. Неоновые вывески. |
| `level2-far.png` | **1920×1080** | Медленная | Зелёная тема. Токсичные крыши. |
| `level2-near.png` | **1920×1080** | Быстрая | Зелёные неоны, индустриальные здания. |
| `level3-far.png` | **1920×1080** | Медленная | Огненное небо. Рушащиеся здания. |
| `level3-near.png` | **1920×1080** | Быстрая | Пламя, лава, руины. |

### Правила для фонов:
1. **Тайлинг:** Левый край картинки должен плавно стыковаться с правым. Проверь: поставь две копии рядом — не должно быть видно шва.
2. **Прозрачность:** Дальний слой (`far`) — полностью залит (включая небо). Ближний слой (`near`) — нижняя часть с прозрачным верхом, чтобы просвечивал дальний слой.
3. **Стиль:** 8-бит пиксель-арт. Москва: высотки Сити, панельки, небоскрёбы. Другие города — свои силуэты.
4. **Ночь vs День:** Для дневных версий — светлое небо, другая цветовая гамма. Можно добавить `level1-far-day.png` и переключать.

### Пример для Москвы (уровень 1):
- **far:** Силуэт Москва-Сити на фоне фиолетового ночного неба. Маленькие жёлтые окна. Звёзды (или нарисуй звёзды, или я оставлю их генерацию в коде).
- **near:** Панельные дома и сталинки на переднем плане. Неоновые вывески "Продукты", "Аптека". Подсвеченные окна.

## 🔧 Как подключить PNG вместо SVG

### Шаг 1: Положи файлы в `assets/sprites/`

### Шаг 2: Отредактируй `js/core/sprites.js`

Замени SVG-функции на загрузку PNG. Пример для зелья:

```javascript
// БЫЛО (SVG):
function createPotion() {
  return svgToImg(`<svg>...</svg>`, 'potion');
}

// СТАЛО (PNG):
function createPotion() {
  const key = 'potion';
  if (cache[key]) return cache[key];
  const img = new Image();
  img.src = 'assets/sprites/potion.png';
  cache[key] = img;
  return img;
}
```

Для ведьм (зависят от темы):
```javascript
// БЫЛО:
function createWitch(theme) {
  return svgToImg(`<svg>...${theme.hairColors}...</svg>`, `witch_${theme.name}`);
}

// СТАЛО:
function createWitch(theme) {
  const key = `witch_${theme.name}`;
  if (cache[key]) return cache[key];
  const img = new Image();
  // Файл зависит от выбранной темы
  const fileMap = {
    'НЕОН': 'witch-neon.png',
    'ТОКСИК': 'witch-toxic.png',
    'ОГОНЬ': 'witch-fire.png',
  };
  img.src = `assets/sprites/${fileMap[theme.name]}`;
  cache[key] = img;
  return img;
}
```

### Шаг 3: Для фонов — отредактируй `js/systems/background.js`

Замени функции `drawFarBuildings` и `drawNearBuildings` на отрисовку PNG:

```javascript
// Добавь в начало файла:
const bgImages = {};

function loadBgImages() {
  for (let i = 1; i <= 3; i++) {
    bgImages[`level${i}_far`] = new Image();
    bgImages[`level${i}_far`].src = `assets/backgrounds/level${i}-far.png`;
    bgImages[`level${i}_near`] = new Image();
    bgImages[`level${i}_near`].src = `assets/backgrounds/level${i}-near.png`;
  }
}
loadBgImages();

// Замени drawFarBuildings на:
function drawFarBuildings(level, time) {
  const img = bgImages[`level${level.id}_far`];
  if (!img || !img.complete) return; // Ещё не загрузилось
  
  const scale = H / img.height; // Подогнать по высоте
  const imgW = img.width * scale;
  const x = (offset * 0.3) % imgW;
  
  // Рисуем два тайла для бесшовного скролла
  ctx.drawImage(img, x, 0, imgW, H);
  ctx.drawImage(img, x + imgW, 0, imgW, H);
  if (x > 0) ctx.drawImage(img, x - imgW, 0, imgW, H);
}
```

## ⚡ Что ОСТАЁТСЯ даже с PNG:
- **Частицы** (хвост от кисти, взрывы при сборе) — генерируются кодом, не зависят от спрайтов
- **Свечение** вокруг кристаллов, щитов, врагов — делается через `ctx.shadowBlur`
- **Scanlines и виньетка** — эффекты поверх всего
- **Звёзды** — можно оставить генерацию кодом или нарисовать на фоне
- **Неоновая сетка на земле** — генерируется кодом
