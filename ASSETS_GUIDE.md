## 1. Общая картина

Сейчас вся графика генерируется кодом (SVG). Ты можешь заменить ЛЮБОЙ элемент на PNG, при этом ВСЕ эффекты (частицы, свечение, тряска) останутся — они рисуются отдельно поверх спрайтов.

## 2. Спрайты

### Общие правила для всех спрайтов
- Формат: **PNG с прозрачным фоном** (alpha канал)
- Стиль: **8-бит / пиксель-арт / кибернеон**
- Размеры указаны в пикселях — делай РОВНО такие, код под них подогнан

### 2.1 Ведьмы (3 штуки)

| Файл | Размер | Что нарисовано |
|------|--------|----------------|
| `witch-neon.png` | **160 × 120 px** | Девушка-ведьма с ФИОЛЕТОВЫМИ волосами на гигантской кисти |
| `witch-toxic.png` | **160 × 120 px** | Та же ведьма с ЗЕЛЁНЫМИ волосами (ядовитый стиль) |
| `witch-fire.png` | **160 × 120 px** | Та же ведьма с ОРАНЖЕВЫМИ/ОГНЕННЫМИ волосами |

Важные детали:
- Ведьма сидит на кисти и **смотрит ВПРАВО** (она летит вправо по экрану)
- Кисть — гигантская кисть для окрашивания волос, с капающей краской
- Слева от кисти оставь 15-20px пустого пространства — там рисуется хвост из частиц
- Шляпа ведьмы — обязательный элемент
- В игре спрайт отображается как 80×60 px (двойной размер для Retina)

Визуальная схема:
```
┌─────────────────────────────────────┐
│                                     │ 120px
│  пусто   [кисть====][ведьма→]      │
│          капли                      │
└─────────────────────────────────────┘
                160px
```

### 2.2 Бонусы (4 штуки)

| Файл | Размер | Что нарисовано | В игре |
|------|--------|----------------|--------|
| `potion.png` | **60 × 80 px** | Розовая баночка профессиональной краски для волос | +10 очков |
| `crystal.png` | **60 × 80 px** | Светящийся голубой кристалл | +50 очков |
| `shield.png` | **64 × 64 px** | Магический щит голубого цвета | Защита 4 сек |
| `heart.png` | **56 × 56 px** | Красное сердце | +1 жизнь |

### 2.3 Враги (3 штуки)

| Файл | Размер | Что нарисовано | Поведение |
|------|--------|----------------|-----------|
| `boxdye.png` | **72 × 80 px** | Коробка дешёвой краски с черепом и "TOXIC" | Летит прямо |
| `scissors.png` | **80 × 72 px** | Раскрытые парикмахерские ножницы | Волнами вверх-вниз |

## 3. Фоны уровней (параллакс)

### Как работает параллакс

Фон состоит из 3 слоёв, каждый движется с разной скоростью:

```
Слой 1 (sky)  — САМЫЙ ДАЛЬНИЙ, почти неподвижный   (×0.1)
Слой 2 (far)  — Дальние здания, медленно            (×0.3)
Слой 3 (near) — Ближние здания, быстро              (×0.6)
─────────── земля (рисуется кодом — неоновая сетка) ──────────
```

### Размеры фонов

| Слой | Размер | Тайлинг | Прозрачность |
|------|--------|---------|--------------|
| `*-sky.png` | **1920 × 1080 px** | Необязателен | НЕТ — полностью залит |
| `*-far.png` | **3840 × 1080 px** | **ОБЯЗАТЕЛЕН** | ДА — верх прозрачный |
| `*-near.png` | **3840 × 1080 px** | **ОБЯЗАТЕЛЕН** | ДА — верх прозрачный |

Почему 3840px для зданий? Это 2 экрана Full HD. Даёт ~15-20 секунд до повтора. Можешь сделать 5760px для ещё меньшего повтора.

### Что такое тайлинг

Левый край картинки должен бесшовно стыковаться с правым:

```
[====картинка====][====картинка====]
                 ↑ тут не должно быть шва
```

Проверка: поставь две копии рядом в Photoshop. Если видно вертикальную полосу — исправь.

### Как рисовать слои

Визуальная схема наложения:
```
┌─────────────────────────────────────────┐
│         sky.png (небо, полная заливка)  │
│                                         │
│    ┌──┐         ┌───┐                   │
│    │  │  far.png│   │ (дальние здания)  │
│    │  │ ┌──┐    │   │                   │
│ ┌──┤  │ │  │ ┌──┤   │                   │
│ │  │  │ │  │ │  │   │                   │
│ │near.png│  │ │  │   │ (ближние здания) │
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓ земля ▓▓▓│
└─────────────────────────────────────────┘
```

- **sky.png** — полностью закрашен, включая небо
- **far.png** — здания в нижних 40-60%, верх прозрачный (видно sky)
- **near.png** — здания в нижних 50-70%, верх прозрачный (видно far и sky)

## 5. Как подключить — изменения в коде

### 5.1 Замена спрайтов

В файле `js/core/sprites.js` заменяем SVG-функцию на PNG-загрузку.

**Было (одна функция, пример для зелья):**
```javascript
function createPotion() {
  return svgToImg(`<svg>... 50 строк SVG ...</svg>`, 'potion');
}
```

**Стало:**
```javascript
function createPotion() {
  const key = 'potion';
  if (cache[key]) return cache[key];
  const img = new Image();
  img.src = 'assets/sprites/potion.png';
  cache[key] = img;
  return img;
}
```

Для ведьм:
```javascript
function createWitch(theme) {
  const fileMap = { 'НЕОН': 'witch-neon.png', 'ТОКСИК': 'witch-toxic.png', 'ОГОНЬ': 'witch-fire.png' };
  const key = `witch_${theme.name}`;
  if (cache[key]) return cache[key];
  const img = new Image();
  img.src = `assets/sprites/${fileMap[theme.name]}`;
  cache[key] = img;
  return img;
}
```

**Не переживай — когда подключишь Claude Code, я сам поменяю нужные функции за тебя.**

### 5.2 Замена фонов

Это более серьёзная переделка `js/systems/background.js`. Вместо процедурных зданий — тайлинг PNG. Я полностью перепишу этот файл когда будут готовы фоны.

### 5.3 Что НЕ надо менять — эффекты которые останутся

| Эффект | Почему не сломается |
|--------|---------------------|
| Хвост частиц от кисти | Привязан к позиции игрока, не к картинке |
| Взрыв при подборе бонуса | Привязан к позиции объекта |
| Свечение вокруг кристалла/врагов | `ctx.shadowBlur` — работает поверх любого спрайта |
| Тряска экрана при ударе | Сдвигает весь canvas |
| Красная вспышка при смерти | Рисуется поверх всего |
| Пульсация сердец | `ctx.scale()` масштабирует любой спрайт |
| Щит-пузырь вокруг игрока | Отдельный круг поверх ведьмы |
| Scanlines и виньетка | Эффекты поверх всего экрана |
| Музыка и звуки | Не связано с графикой |
| Неоновая сетка на земле | Генерируется кодом, остаётся |

## Быстрый старт
